# 2026-05-10

## App Server schema tools

今日は runtime session store の上に App Server tool 境界を追加する。対象は
`app_server_schema_tools.mbt` と公開型を置く `domainprocessschema_mbt.mbt`、確認用の
`app_server_schema_tools_test.mbt` に絞る。

read-only、mutating、approval-required を型で分け、HTTP や永続化 adapter は入れない。
approval-required は実行結果ではなく approval proposal、つまり client が承認 UI に渡す
ための提案データを返す形にする。
