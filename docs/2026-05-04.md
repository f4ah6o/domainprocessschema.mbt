## 学んだこと

- **SSOT** は Single Source of Truth の略で、同じ情報を何度も別形式で持たず、一つの定義から複数の成果物を生成する考え方を指す
- 今回はいきなり API や GUI まで広げず、まず SQL という具体的な生成結果に絞ることで、schema IR の妥当性を検証しやすくした
- MoonBit では package ではなく directory が実際の compilation unit なので、同じ package の中なら file を分けても一つの実装として扱える
