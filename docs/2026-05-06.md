## 学んだこと

- manifest envelope を入れると renderer だけでなく consumer 側 parser も同時に直さないと、`payload` を見つけられずに壊れる
- structured diagnostics は type を足すだけでは足りず、CLI / browser host / tests の「表示面」まで一緒に直して初めて価値になる
- `schema contract` を先に文章化すると、どこまでが stable でどこからが follow-up かを README や issue から辿りやすくなる
